# YOUDY 📺📚

**YOUDY**는 **유튜브를 활용한 학습 일정 및 목표 달성 웹 사이트**입니다.  
학습용 유튜브 영상을 일정에 따라 관리하고, 목표를 시각적으로 확인할 수 있도록 도와줍니다.

---

## 🔧 기술 스택

- ⚛️ **React** 
- 🌐 **Next.js**
- 🎨 **Tailwind CSS**
- ⛑️ **TypeScript**
- 🧩 **shadcn/ui**

---

## 🚀 주요 기능

- 📅 **학습 일정 등록 및 관리**
- 📽️ **유튜브 영상 연동 및 학습 진행률 체크**
- 🧠 **진행 상태 시각화 (진행률 바, 캘린더)**
